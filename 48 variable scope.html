JS has two scopes:- 
        - Global
        - Local

            Global:-
            variable declared outside function.
            it's value is accessible/modifiable throughout the program.
            these are availble to new pages loaded in the same browser window.

            Local:-
            variable declared inside function.
            gets created & destroyed every time function is executed.
            can't be accessed outside function.
            NOTE:- if inside function a variable is not declared with "var", 
                it is treated as global variable.

            Local:- 
            if a function within a function, inner function can access
            outer function's variables but outer function can't access
            inner function's variables.
